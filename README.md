# Bilustek Homebrew Tap

Homebrew formulae for Bilustek CLI tools.

## Usage

```bash
brew tap bilustek/tap
```

## Available Formulae

| Formula | Description |
|:--------|:------------|
| `gosecrets` | Encrypted credentials management CLI tool |

### Install

```bash
brew install gosecrets
```

### Update

```bash
brew update
brew upgrade gosecrets
```
